# 国际·中国 产业金融雷达

一套自建的产业金融情报终端，聚焦 **物理AI（具身智能）** 主线，把「全球事件 → 中国产业 → 产业链环节 → 上市公司实时行情 → 未上市早期公司」串成一条可下钻的链路。

**零外部依赖**：纯 Node.js 内置模块 + 原生前端（无框架、无构建、无 npm install）。

## 物理AI专区（核心）

项目的战略主线。位于「硬科技初创」标签顶部，是一个覆盖 **物理AI / 具身智能全产业链** 的专业终端。

### 三层 20 个细分环节

```
上游 · 核心零部件
  灵巧手 · 触觉/力矩传感 · 无框力矩电机 · 空心杯/无刷电机 · 谐波/RV减速器
  行星滚柱丝杠 · 编码器/轴承 · 六维力传感 · 电子皮肤/柔性传感
  轻量化材料PEEK · 机器人芯片/域控

中游 · 本体 + 具身大脑 + 仿真
  人形机器人本体 · 关节模组/一体化执行器 · 四足/移动机器人
  工业/协作机械臂 · 具身大模型VLA · 仿真/数据/空间智能

下游 · 应用场景
  工业/智能制造 · 服务/商用 · 物流/仓储
```

### 数据设计

- **上市供应商**：绿的谐波、双环传动、兆威机电、埃斯顿、鸣志电器、三花智控等 **30+ 家**，全部走 `/api/cn-stocks` **实时行情**（红涨绿跌，点击跳东财），每 30 秒刷新。所有股票代码均已校验可解析。
- **未上市新锐**：宇树、智元、银河通用、自变量、穹彻、星海图、松延动力等，标注阶段徽章（独角兽 / 拟IPO / 明星 / 早期 / 港股）。
- **点任一环节** → 弹出该环节详情：环节内全部企业（上市带实时行情）+ 该环节实时快讯。
- **专区内嵌**：物理AI 实时融资流 + 物理AI 政策与事件（策展政策 + 实时政策快讯）。

### 早期公司挖掘的方法论

早期公司大多未上市、没有行情数据，因此**不做大规模人工策展**（不可信、不可扩展），而是**从实时融资快讯里挖掘**：

- **两级过滤**：命中强早期轮次（种子/天使/Pre-A/A/A+/A1轮）**或**明确融资动作 + 成果转化；
  硬排除 B 轮及以后、Pre-IPO、上市过会、二级市场、债权授信、基金设立、中标签约等非早期股权融资。
- **按公司去重**：提取融资动词前的公司名做去重，合并同一家公司被多家媒体报道的重复条目。
- 该过滤同时服务于：早期雷达 / 融资雷达「仅早期」/ 省份新锐三处。

## 快速开始

**要求：Node.js ≥ 22.5**（使用了内置 `node:sqlite`）。无需 `npm install`。

```bash
cd 金融雷达/global-app
node server.js
```

打开：

- 中国雷达 <http://127.0.0.1:4173/china.html>
- 产业链雷达 <http://127.0.0.1:4173/chain.html>
- 全球雷达 <http://127.0.0.1:4173/index.html>（需外网）
- 后台管理 <http://127.0.0.1:4173/admin.html>

自定义端口：`PORT=8080 node server.js`
数据目录（用户库与配置的落盘位置）：`DATA_DIR=/var/data node server.js`

> Windows 可直接双击 `金融雷达/global-app/start-live.cmd`。
> 修改前端静态文件（`*.js` / `*.css` / `*.html`）**免重启**，刷新即可；改 `server.js` 需重启。

## 数据源与实时性

| 雷达 | 数据源 | 无外网（仅国内网络） |
|---|---|---|
| 中国雷达 | 东方财富、新浪财经；地图 GeoJSON 走阿里云 DataV | ✅ **完全实时可用** |
| 产业链雷达 | 东方财富行情接口 | ✅ **完全实时可用** |
| 全球雷达 | GDELT、Yahoo Finance、Google News、国际 RSS；地图瓦片走 CartoDB | ❌ 抓取失败会**回退演示数据**，地图瓦片加载不出 |

**结论**：在仅有国内网络的环境下，中国雷达 + 产业链雷达 + 物理AI专区可直接上线使用；全球雷达需要外网，否则只能看演示数据。

行情刷新节奏：物理AI供应商与省域代表企业 30 秒；融资/政策/快讯 2 分钟。

## AI 功能配置

AI 相关能力（每日简报、智能问答、新闻→板块影响分析、融资快讯提炼初创公司榜单）由 **DeepSeek** 驱动，**可选**——不配置时其余功能全部正常，AI 区块会优雅降级为提示文案与原始快讯流。

两种配置方式（后台配置优先）：

1. **后台**：打开 `admin.html` →「系统管理」→ 填入 DeepSeek API Key（保存到 `DATA_DIR/radar-config.json`，重启不丢）
2. **环境变量**：`DEEPSEEK_API_KEY=sk-xxx node server.js`

配置后，早期雷达与融资雷达顶部会自动把快讯提炼成结构化榜单：**公司 · 轮次 · 金额 · 赛道**（同一公司自动合并，优先早期轮次，物理AI赛道细分到具身VLA/人形/灵巧手/减速器/丝杠/机器人电机等）。

后台可按功能粒度开关：`brief` / `chat` / `analyze` / `insight` / `startups`。

## 目录结构

```
金融雷达/
├── global-app/                 # 应用主体
│   ├── server.js               # 纯 Node HTTP 服务 + 全部 API
│   ├── auth.js                 # 账号系统（node:sqlite）
│   ├── config.js               # 运行时配置（DeepSeek key / 功能开关）
│   ├── archive.js              # 新闻归档
│   ├── translate.js            # 标题翻译
│   │
│   ├── china.html / china-app.js / china-app.css    # 中国雷达（含物理AI专区）
│   ├── chain.html / chain-radar.js / chain-radar.css # 产业链雷达
│   ├── index.html / app.js / upgrade.js / styles.css # 全球雷达
│   ├── admin.html / admin.js   # 后台管理
│   │
│   ├── data/                   # 静态参考数据（海关/运价/信保）
│   ├── deploy/                 # 部署脚本与文档
│   └── artifacts/              # 界面截图
└── vendor/                     # 本地化的 Leaflet 等前端库
```

主要 API：`/api/cn-stocks`（行情）、`/api/cn-search`（快讯搜索）、`/api/cn-startups`（融资快讯 AI 提炼）、`/api/snapshot`（全球快照）。
