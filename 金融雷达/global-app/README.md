# GeoTrade Radar MVP

一个独立实现的全球地缘政治、贸易、供应链与金融市场风险仪表盘原型。
产品形态参考 World Monitor 的信息组织思路，但未复制其源码。

## 运行

双击 `start-live.cmd`，浏览器会打开 `http://127.0.0.1:4173`。

也可以使用 Node.js 手动运行零依赖实时数据服务：

```powershell
node server.js
```

然后打开 `http://127.0.0.1:4173`。

直接打开 `index.html` 时仍可浏览演示数据；真实数据同步需要本地实时服务处于运行状态。

## 真实数据源

- 全球事件：GDELT 全球新闻事件流；GDELT 限流时自动切换 Google News 与多家公开 RSS
- 产业企业：从 GDELT 实时事件流中抽取科技产业与代表企业动态；事件流缺失时自动切换公开科技 RSS 与 Google News RSS 兜底
- 市场行情：Yahoo Finance；不可用时自动切换欧洲央行官方汇率与 CoinGecko 行情
- 官方制裁：美国财政部 OFAC Recent Actions

市场行情每 60 秒同步，全球事件与产业企业动态每 5 分钟同步，OFAC 官方制裁每 15 分钟同步。来源不可用时，界面会明确显示“部分实时”“陈旧数据”或“演示数据”。

## 当前能力

- 真实全球事件、市场行情与 OFAC 制裁同步，并保留结构化演示数据作为离线降级
- 日、周、月时间范围筛选与手动刷新
- 事件筛选、搜索、地图点位联动，并可从事件卡片或详情栏打开原始报道
- 可缩放真实地理底图，支持事件、贸易路线、关键海峡和港口图层；网络不可用时自动使用离线矢量兜底地图
- 今日全球风险总览、路线压力排行与事件“为什么重要”分析
- 实时关键信号由当前事件与行情派生；路线和国家风险模块明确标注为参考模型
- 受影响国家、行业、商品、路线、市场资产和后续跟踪指标
- 观察列表与高风险筛选
- 市场雷达、贸易政策、供应链、制裁追踪、AI 简报
- AI 与机器人产业情报雷达，支持算力、机器人、政策合规和投资机会筛选，并可跳转关联全球事件
- 物理AI专题雷达，跟踪具身模型、仿真训练、边缘推理、安全合规和现场运营闭环
- 产业企业实时动态，覆盖 AI、机器人、物理AI、算力芯片、云与数据中心、网络安全、智能汽车与电池、量子科技、航天防务、消费电子、生物科技和金融科技，并保留原文链接
- 国家—行业矩阵、机会雷达和情景推演
- 默认超大阅读字号，并提供 `A− / A+` 阅读字号调节
- 根据屏幕宽度与高度自动适配：大屏三栏、中屏事件抽屉、平板与手机单列布局
