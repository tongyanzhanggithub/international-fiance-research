# 更新日志 Changelog

本项目版本遵循语义化版本（SemVer）。日期为北京时间。

## [1.0.0] - 2026-06-14

首个正式版本：6 个雷达的完整情报平台，完成安全加固，并具备一键云部署能力。

### 新增
- **雷达矩阵（6 个）**：全球贸易风险、华贸（中国）、供应链与航运、能源与大宗商品、国际产业、制裁与合规。
- **制裁与合规雷达**：复用 OFAC Recent Actions 实时动态 + 合规参考模型（制裁项目 / 清单筛查 / 红旗信号 / 筛查清单）。
- **国际产业雷达**：33 产业 × 8 大类，国内/国外对比 + 中国地位 + 自主可控（卡脖子）榜。
- **供应链与航运雷达**：SCFI 实时运价 + 航线压力 + 关键节点。
- **能源与大宗商品雷达**：16 商品（能源/贵金属/工业金属/农产品）Yahoo 报价 + 降级。
- 顶栏新增可见的“全部雷达”返回按钮（网格图标，位于品牌左侧）。

### 实时化（去除“做样子”的演示数据）
- 各子雷达顶栏状态徽章改为由实时快照计算（出口增速、运价涨跌、制裁动态数、自主可控度等）。
- 航运“供应链事件”、能源“大宗事件”接入 `/api/snapshot` 真实 GDELT/公开新闻（按 supply/energy 分类），空时回退示例并标注。

### 安全
- 修复全球雷达事件流/详情/影响链/AI 简报/市场卡的 XSS（外部数据未转义）。
- 静态服务路径穿越防护 + 敏感文件黑名单（users.db / 备份 / 点文件不可下载）。
- 雷达数据端点统一登录守卫 + 按 IP 限流。
- 管理口令改为占位符（仓库不含真实密钥）。

### 部署
- `Dockerfile` + `.dockerignore`：零依赖 Node 24 镜像，内置健康检查。
- `render.yaml`：Render 一键蓝图（持久磁盘 + 海外节点 + 密钥环境变量）。
- 用户库/缓存路径支持 `DATA_DIR` 环境变量，适配持久磁盘。
- `deploy/部署到云平台.md`：Render / Docker / VPS 三方案指南。

### 修复
- `refresh-data` 双处理器冲突、华贸趋势空数组、能源 topLosers 排序、华贸出口增速取数错误等。
